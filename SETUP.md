# OvenMediaEngine 與 Sinatra 管理介面安裝指南

## 使用 Docker 安裝

1. 確保您的系統已安裝 Docker 和 Docker Compose。

2. 克隆專案到本地：

   ```
   git clone <your-repo-url>
   cd <your-repo-directory>
   ```

3. 構建並啟動容器：

   ```
   docker-compose up --build
   ```

4. 訪問管理介面：
   在瀏覽器中打開 `http://localhost:4567`
